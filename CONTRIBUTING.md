# Contributing to Gravity Stream Orchestrator

First off, thank you for considering contributing to Gravity! It's people like you that make this a great tool for the AI community.

## 🌈 How Can I Contribute?

### Reporting Bugs
- Use the GitHub Issue Tracker.
- Describe the bug and include steps to reproduce it.
- Mention your Rust version and OS.

### Suggesting Enhancements
- Open an issue describing the feature.
- Explain why it would be useful.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the test suite passes (`cargo test`).
4. Format your code with `cargo fmt`.
5. Run clippy to ensure best practices: `cargo clippy`.
6. Submit the pull request!

## 📜 Style Guidelines
- Follow [Standard Rust Style](https://doc.rust-lang.org/1.0.0/style/README.html).
- Write descriptive commit messages.
- Add documentation for any new public modules or structs.

## 🛠 Development Setup
1. Install Rust via `rustup`.
2. Clone your fork.
3. Run `cargo build` to ensure dependencies are resolved.

## 🧪 Testing
We aim for high test coverage for the protocol and routing logic.
```bash
cargo test
```

Thank you for your help!
